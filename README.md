# Project Tactics

Run (with hot-reload):
```shell
./run_game.sh
```
Build (hot reload):
```shell
./build_game_hot.sh
```

Build (-debug):
```shell
odin build ./src/game -build-mode:dll -out:game.bin -debug ; odin run ./src/tactics.odin -file -debug
```
